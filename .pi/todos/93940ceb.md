### Features
- MutationObserver detects Bases table views and injects preview column
- Reads note content via `app.vault.cachedRead()`
- Settings tab: preview length, content source (full body / heading), heading name, show add button toggle
- "+ New note" button below each Base table creates a file with matching frontmatter
- Hover-expand on preview cells
- Debounced observer to handle Bases dynamic re-rendering
